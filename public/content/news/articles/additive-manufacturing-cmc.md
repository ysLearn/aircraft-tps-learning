# 西工大在3D打印陶瓷基复合材料防热构件方面取得新进展

## 研究背景

SiC/SiC陶瓷基复合材料具有耐高温（>1400°C）、低密度、抗氧化等优异性能，是新一代航空发动机和高速飞行器热防护结构的理想候选材料。然而，传统制备工艺存在周期长、成本高、难以成形复杂构件等问题。

## 技术突破

西北工业大学凝固技术国家重点实验室利用**数字光处理（DLP）3D打印技术**成功制备出复杂几何形状的C/SiC和SiC/SiC防热构件。

### 工艺路线

1. **光敏浆料配制**：SiC粉末与光敏树脂混合，优化分散剂含量
2. **DLP打印成形**：层厚50 μm，曝光时间3-5秒/层
3. **脱脂处理**：惰性气氛下缓慢升温至800°C去除有机物
4. **熔渗致密化**：液态硅渗透（LSI）工艺填充残余孔隙

### 性能指标

| 性能 | 数值 |
|------|------|
| 致密度 | 理论密度的 **97%** |
| 弯曲强度 | **> 400 MPa** |
| 断裂韧性 | **4.2 MPa·m¹/²** |
| 1500°C弯曲强度保持率 | **85%** |

## 典型构件

团队成功打印了**涡轮叶片冷却通道、燃烧室浮动瓦块、前缘驻点帽**等多种典型防热构件，最大尺寸达到150 mm，壁厚最薄处仅0.5 mm。

## 应用前景

- 航空发动机热端部件快速原型制造
- 高超声速飞行器异形TPS构件制造
- 冷却结构的一体化设计与制造

> "DLP增材制造技术为复杂形状陶瓷基防热构件提供了全新的制造范式，有望大幅缩短研发周期和降低成本。"
